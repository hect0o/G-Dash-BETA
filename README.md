# G-Dash BETA

A Modern, minimal tablet like looking interface for Geo-Fs. This project is in beta now, so watch out for some bugs😭

## Features
*Live Map with all txy etc
*Logbook
*Accurate Flight timetracker
*Weather Map (Coming Soon)
*Etc

### Step 1
